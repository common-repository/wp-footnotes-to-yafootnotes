Automatically changes footnote syntax from WP-Footnotes to YAFootnotes Plugin.

== Description ==

Automatically changes footnote syntax from WP-Footnotes to YAFootnotes Plugin on the fly. One can change syntax permanently by going to "Settings->Change Footnotes". This plugin is created for Darkmatter journal.

== Installation ==

1. Upload `changeFootnotes.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. (optional): Go to "Settings->Change Footnotes" and, by clicking "Change Syntax", change syntax permanently.

== Frequently Asked Questions ==

= Can I disable this plugin after changes are done? =

Yes, after changing syntax permanently ("Settings->Change Footnotes"), you can disable this plugin as it doesn't do anything else.
